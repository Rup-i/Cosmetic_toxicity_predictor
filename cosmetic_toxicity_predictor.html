Methodology

From molecule to prediction

A four-stage cheminformatics pipeline converts any chemical structure into a probabilistic safety assessment.

🧪
SMILES Input
Canonical SMILES string encoding the full molecular structure and connectivity.
🔬
RDKit Parsing
RDKit validates and normalizes the molecule, rejecting invalid structures gracefully.
🧮
Morgan Fingerprints
2048-bit binary fingerprint (radius=2) captures circular substructure patterns around each atom.
⚖️
Ensemble Classifier
RF or XGBoost outputs a calibrated toxicity probability, then thresholded for a final verdict.

Threshold Calibration (XGBoost)
Lowering the decision threshold to 0.25 sacrifices some precision to dramatically improve toxic recall — the right trade-off for a safety tool.
0.0 — Classify all as toxic 0.25 ← Conservative 0.50 — Default 1.0 — Classify all as safe
✓ Benefit — Lower Threshold
Higher recall on toxic class. Fewer dangerous compounds are missed. Critical for consumer safety applications.
✗ Cost — Lower Threshold
More false positives. Safe compounds get flagged, increasing unnecessary review burden.
⚖ Recommended Setting
0.25 — Chosen as the default for cosmetic screening, where missing a toxin is far more costly than a false alarm.
ℹ Domain Context
ClinTox labels clinical trial failures and FDA-approved drugs. Threshold tuning maps lab predictions to real-world safety decisions.
